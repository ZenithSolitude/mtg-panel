# MTG Panel v2 — Web Management Dashboard

Современная веб-панель для управления MTProto прокси-сервером [mtg](https://github.com/9seconds/mtg).

## ⚡ Установка одной командой

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/ZenithSolitude/mtg-panel/main/install.sh)
```

## 🆕 Новое в v2

| Функция | Описание |
|---|---|
| 📱 **QR-код** | Сканируйте камерой телефона — Telegram открывается автоматически |
| 🔗 **Счётчик соединений** | Видно сколько клиентов подключено прямо сейчас |
| 🔔 **Telegram алерты** | Уведомление в личку при падении/восстановлении прокси |
| 🛡️ **IP Whitelist** | Разрешить вход в панель только с ваших IP/CIDR |
| 📋 **История событий** | Лог всех запусков, остановок, крашей и обновлений |
| 💾 **Диск + аптайм** | Метрики диска и время работы службы |
| 🎨 **Новый дизайн** | CRT/терминальная эстетика, 5 вкладок |
